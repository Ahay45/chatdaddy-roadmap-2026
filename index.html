Product-Led Growth

— The product itself drives acquisition, activation & expansion

PLG flips the funnel. Instead of sales pushing users to the product, the product pulls them in, shows value before a credit card appears, and turns power users into advocates. Every feature below is a growth lever baked into the UX itself.

PLG

Freemium "Taste of Power" Tier

Let any team send up to 500 messages/month free — no credit card. The moment they hit the limit they're already hooked.

  • Usage meter visible in header (creates urgency)
  • Smart upgrade prompt at 80% usage, not 100%
  • One-click upgrade preserves all data & flows
Q2 2026
PLG

Viral Invite Loop

Every message sent via ChatDaddy carries a subtle "Powered by ChatDaddy" badge for free-tier users — clickable, leading to a pre-filled signup.

  • Badge opt-out unlocked at paid tier (upgrade incentive)
  • Referral credit: invite a team → get 1 month free
  • Referral dashboard showing earnings & clicks
Q2 2026
PLG

In-app "Aha Moment" Funnel

Data shows the aha moment is sending a first automated reply. Track time-to-first-automation and optimise every step before it.

  • Funnel: signup → connect channel → first flow → first reply
  • Drop-off heatmap per step (Amplitude)
  • Automated nudge email + in-app toast at each stuck step
Q1 2026
PLG

Template Marketplace (Virality)

Users publish their best flows as public templates. Others install with one click — every install credits the creator and promotes ChatDaddy.

  • "Built with ChatDaddy" share card for social
  • Top templates featured on landing page (SEO)
  • Creator badges + profile page
Q4 2026
PLG

Usage-Triggered Expansion Revenue

Identify power users automatically and surface upgrade offers contextually — inside the feature they love most, right when they hit a limit.

  • AI model scores each team's expansion readiness
  • Personalised upgrade CTA (e.g. "You sent 480 messages this week — upgrade to unlimited")
  • CSM alert for high-score teams
Q3 2026
PLG

Public ROI Calculator

Embeddable calculator on marketing site: "How many hours does your team spend on manual WhatsApp replies?" → shows $ saved with ChatDaddy.

  • Inputs: team size, avg reply time, messages/day
  • Output: time saved + cost saved per month
  • CTA: "Start saving now" → pre-filled signup
Q2 2026

UIUX Onboarding Experience

— Signup-level & feature-level

Bad onboarding is the #1 reason users churn before they ever experience value. We fix it at two levels: Signup-level (first 5 minutes) and Feature-level (every time a user touches something new).

① Signup-Level Onboarding

Onboarding

Zero-Friction Signup Flow

Reduce signup form to email + password only. Everything else is collected progressively after the user is inside.

  • Google / WhatsApp OAuth as primary CTA
  • No credit card on free tier
  • Auto-create a demo workspace so they land in a live product
Q1 2026
Onboarding

Role-Based Welcome Path

"What best describes you?" — 3 cards (Business Owner / Support Agent / Developer). Each path shows a different first-run experience tailored to their goal.

  • Business Owner → Revenue & automation focus
  • Support Agent → Inbox & team tools focus
  • Developer → API & FlowBuilder focus
Q1 2026
Onboarding

Interactive Setup Checklist

Sticky sidebar checklist that tracks activation milestones. Gamified with progress ring and confetti on completion.

  • Connect WhatsApp channel
  • Invite first teammate
  • Send first message or automation
  • Dismiss only when all 5 steps done
Q1 2026
Onboarding

Guided Channel Connection Wizard

Replace the current broken WABA OAuth flow with a step-by-step modal with live status indicators and clear error recovery.

  • Step indicators: Auth → Verify → Connect → Done
  • Inline error messages (no "Oops!" page)
  • Video thumbnail for each step
Q1 2026
Onboarding

Day-1 Drip Nudges

Automated in-app + email nudges triggered by what the user has NOT done yet — not generic blasts.

  • 2h after signup if no channel connected → "Need help connecting?"
  • 24h if no message sent → short video tutorial
  • 72h if no teammate invited → "Better together" prompt
Q2 2026

② Feature-Level Onboarding

Onboarding

Contextual Spotlight Tooltips

First time a user visits FlowBuilder, CRM, Campaigns — a spotlight highlights the key action with a single sentence. Dismissed in one click.

  • Shown only once per feature per user (persisted)
  • Skip all option for power users
  • Tooltip copy written at reading level 8 (no jargon)
Q2 2026
Onboarding

Empty State Design System

Every empty state (no chats, no flows, no campaigns) tells the user exactly what to do next — with an illustration and a primary CTA.

  • Replace all blank white screens
  • CTA directly creates the first item (not a link to docs)
  • Consistent illustration style across all features
Q1 2026
Onboarding

Interactive Product Tours

Optional "Take a tour" button on complex features (FlowBuilder, CRM, Analytics). Walks through 4–5 steps with real UI interaction.

  • Tours built with Shepherd.js (lightweight, no third-party)
  • Resumable — user can pause and continue later
  • Completion tracked in Amplitude for ROI measurement
Q2 2026
Onboarding

Starter Templates Per Feature

Every new flow, campaign, or chatbot starts with a "Use a template" option — pre-filled with realistic example content.

  • 5 curated templates per feature category
  • Preview modal before inserting
  • Editable immediately after insert
Q2 2026
Onboarding

In-app Help Panel

Slide-out panel (?) icon in every feature — shows 3 contextual help articles, a video, and a "Chat with support" button.

  • Context-aware: different content per page
  • Searchable knowledge base inline
  • Escalates to live chat if article doesn't resolve
Q3 2026
Onboarding

Feature Adoption Scorecards

Internal dashboard (for CSM & product team) showing per-team adoption depth across all features — so we can proactively reach struggling teams.

  • Score: 0–100 per feature per team
  • Auto-flag teams stuck at <30% after 14 days
  • CSM outreach queue sorted by churn risk
Q3 2026
